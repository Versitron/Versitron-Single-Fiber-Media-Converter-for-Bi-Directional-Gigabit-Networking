## What is a Single Fiber Media Converter?

A single fiber media converter enables full-duplex Ethernet transmission over one optical fiber strand. Instead of using two fibers for separate transmit and receive paths, it combines TX and RX traffic through wavelength-based separation. This design supports efficient network expansion while maintaining reliable data performance.
